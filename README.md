# Event Management System with AI Chatbot

A modern, interactive web application for discovering, registering, and managing tech events with an integrated AI-powered chatbot assistant.

## Features

### Event Discovery
- Browse upcoming tech events worldwide
- Search functionality to find specific events
- Detailed event pages with comprehensive information

### Event Registration
- User registration system with name and email
- Event selection dropdown
- Confirmation system for successful registration

### Organizer Dashboard
- Add new events to the system
- Manage event details including name and date

### Chatbot Assistant
- Intelligent chatbot to help users navigate the platform
- Answer questions about events, registration, and event details
- Assist with common inquiries about event schedules, locations, and prerequisites

### Event Details
- Comprehensive event descriptions
- Curriculum breakdown for workshops
- Hands-on lab information
- Date, time, and location information
- Related events recommendations

### suggestions based on your own search

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Storage:** Local storage

## Prerequisites

Before you begin, ensure you have nodejs ans npm installed:
### for windows:
1. Download Node.js from their official website: https://nodejs.org/
2. Run the installer
3. **Important:** During installation, make sure "Add to PATH" is checked (it is by default)
4. Restart your computer
5. The PATH is automatically configured by the installer 
### arch linux:
-run the command sudo pacman -S nodejs npm
### for kali :
-sudo apt update
-sudo apt install nodejs npm

### run to confirm that u have the latest version of both
-node --version
-npm --version
### use this command to run the web
-node app/app.js
